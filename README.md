## ✨ Features

- **SHA-256 Idea Fingerprinting** — Unique cryptographic hashes create immutable proof of existence
- **AES-256-GCM Encryption** — Military-grade layer encryption for secure idea disclosure
- **5-Layer CID Vault** — Progressive disclosure system for sharing ideas safely
- **Innovation Score Engine** — AI-powered scoring for idea valuation
- **Escrow Payment Hold** — Secure transactions with fund protection
- **Legal IP Certificates** — Official PDF certificates for ownership proof
- **DocuSign NDA Integration** — Built-in legal protection before sharing

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React + Heroicons
